✅ Survivorship-free

Survivorship-free data: prices come from NSE daily bhavcopies, which include every stock that traded each day — delisted/dead names too, so the backtest is not survivorship-biased. Positions in a stock that stops trading are marked to zero (the loss is realised, not hidden). The universe is filtered by point-in-time daily turnover or point-in-time index membership, so there's no size look-ahead either. Prices are split/bonus-adjusted; returns are price-only (no dividends), before costs/slippage/tax. Not investment advice.
